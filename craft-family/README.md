# Craft Family Design System

Craft Family (CFDS) 是一个统一的设计系统，覆盖效率工具与生活助手两大系列应用。

## 产品架构

- **Efficiency Series**: FlashCraft (记忆), NoteCraft (笔记), MarkCraft (写作), CourseCraft (教学管理)
- **Life Series**: WeatherCraft (气象)

## 视觉基础

- **Craft Black** `#171717` — 主按钮、Logo 背景
- **Craft White** `#ffffff` — 卡片背景
- **Craft Gray** `#f5f5f7` — 全局背景
- **Craft Yellow** `#facc15` — 品牌识别色
- **Craft Border** `#e5e5e5` — 极细边框

## 核心交互

- **Blur-Clear Entrance**: 从模糊到清晰的进场动画
- **Gentle Float**: 元素呼吸漂浮感
- **Micro-interactions**: 按钮点击缩放、成功反馈

## 技术栈

React 18 + Tailwind CSS + Lucide Icons + Inter / JetBrains Mono 字体

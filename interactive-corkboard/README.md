# Interactive Corkboard

一个可拖拽、可连接的交互式软木板。把便签、拍立得、工牌和笔记本纸条钉在软木板上，再用线把它们连起来。

## 交互

- **拖拽移动**：按住任意卡片拖动
- **缩放与平移**：鼠标滚轮缩放画布，空白处拖拽平移
- **连接模式**：点击卡片边缘的连接点，再点击另一个卡片的连接点即可连线
- **编辑内容**：双击便签上的文字可直接编辑
- **底部工具栏**：添加新便签、重置视图、切换连接模式

## 视觉

- 矢量软木纹理背景
- 真实图钉、胶带、工牌夹
- 四种便签颜色 + 撕边笔记本纸条
- 拍立得照片框
- 手写风格字体点缀

## 运行

直接打开 `preview.html` 即可。推荐用本地服务器避免跨字体限制：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173/interactive-corkboard/preview.html`。
